Open Archaeology Data Search

Open Archaeology Data search is a search engine build by the Open State Foundation to search through all the data in the Open Archaeology Data API. Open Archaeology Data Search is a single page application (SPA) using the front-end Angular framework. Open Archaeology Data Search relies on Bootstrap for the interface components and the responsiveness.

Open Archaeology Data

Open Archaeology Data was developed as a part of the Tandem Europe Cultural Managers Exchange Programme in a collaboration by the Open State Foundation in the Netherlands and Virtutim in Italy.

Open Archaeology Data API

The Open Archaeology Data API was launched in 2017. This API offers an infrastructure to collect, search and re-use combined open archaeological datasets of European archaeological organisations.

The Open Archaeology Data API is based on an Elasticsearch search engine. This open source technology is used to centrally store the datasets and to make all the data searchable in a universal manner. An extract, transform, load system is used, through which all the different datasets are harvested (the original data always remains available in the API), adjusted, (to support the universal searchability) and eventually saved.
